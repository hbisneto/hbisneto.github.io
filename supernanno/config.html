Declarative Config

.supernannorc

Simple, Nano-style configuration. Parsed by services/rc_parser.py and applied live by the Async Config Watcher.

# SuperNanno configuration

set backup
set backupdir ~/.config/Bisneto/SuperNanno/Backup
set configwatcher
set configwatcherinterval 1
set indenttype spaces
set operatingdir ~/
set pathdisplay full
set restoresession
set sidebar
set sidebarwidth 35
set tabbehavior indent
set tabsize 4

# Disable option:
unset backup
