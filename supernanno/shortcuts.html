Core Shortcuts

File & Editor

CTRL+N New file Create a new empty buffer
CTRL+O Open file Open existing file (path input)
CTRL+S Save file Save current buffer (atomic write)
CTRL+R Insert file at cursor Insert content from another file
CTRL+Q Quit Exit editor (with unsaved check)
CTRL+B Toggle Sidebar Show/hide file explorer
F1 Open Settings Settings screen (grid UI)
CTRL+X Generate Report Create GitHub diagnostic issue
Search Engine

Search & Replace

CTRL+F Find / Search Open search bar
CTRL+H Find & Replace Open search with replace mode
Enter Next match
Shift+Enter Previous match
↓ Enter Replace mode
↑ Return to Find mode
CTRL+H Replace All Replace all matches at once
ESC Close search
Settings Screen

F1 — Settings

CTRL+S Save Settings
ESC Close Settings Return to editor
