Command Line

Basic Usage

Full parser implemented in cli/parser.py + cli/models.py

General Syntax
$ supernanno [OPTIONS] [[+LINE[,COLUMN]] | +/SEARCH] [FILE]
Common Examples
$ supernanno
# Open new buffer (or restore session)
$ supernanno my_file.py
$ supernanno +42 main.py
# Jump to line 42
$ supernanno +/TODO app.py
# Open and search for "TODO"
Advanced Flags
$ supernanno -v config.json
# Read-only mode
$ supernanno -B file.py
# Enable automatic backups
$ supernanno -B -C ~/Backups/ project.py
# Backup to custom directory
Reference

All CLI Options

Available Arguments & Flags
FILE str | None Path to open (relative or absolute)
+LINE int Jump to specific line on startup
+LINE,COL int,int Jump to line and column
+/SEARCH str Open file and start search with term
-v, --view bool Read-only mode
-B, --backup bool Enable automatic .bak backups before save
-C, --backupdir <dir> str Custom backup directory (~ expansion supported)
-h, --help bool Show help and exit
-V, --version bool Show version and exit
