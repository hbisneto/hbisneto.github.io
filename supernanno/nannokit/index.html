NannoKit

A modern, lightweight set of file and message dialogs built for Textual applications.

NannoKit

The Official Component Suite

NannoKit é a suíte oficial de bibliotecas e componentes reutilizáveis para aplicações Textual. Cada pacote é modular, bem documentado e projetado para funcionar tanto dentro do SuperNanno quanto em qualquer outro projeto TUI.

NannoKit.Dialogs
Diálogos modernos e robustos inspirados no .NET. Inclui OpenFile, SaveFile, OpenFolder, messagebox e sistema inteligente de prioridade e isolamento entre diálogos.
Estável
NannoKit.Editor
Componentes centrais de edição: AdvancedTextArea, suporte completo a Tree-sitter, múltiplos cursores, busca/substituição avançada, word wrap, auto-indent e infraestrutura de plugins.
Em Desenvolvimento
NannoKit.Terminal
Ferramentas avançadas de terminal: execução de comandos, painéis de output, integração com shell, emulação de terminal, captura de saída e gerenciamento de processos.
Planejado
NannoKit.GetInfo
Utilitários para obter informações do sistema, arquivos, metadados, processos e ambiente de forma consistente e com tratamento robusto de erros.
Planejado
NannoKit.LSP
Suporte ao Language Server Protocol. Autocomplete, diagnósticos em tempo real, go to definition, hover information e formatação inteligente para múltiplas linguagens.
Planejado
NannoKit.Settings
Sistema completo de configurações com interface gráfica, validação de campos, persistência segura, watcher em tempo real e suporte a temas.
Planejado
NannoKit.Plugins
Sistema de plugins poderoso com leader mode, bindings customizáveis, metadata, auto-descoberta e sandbox seguro para extensões.
Planejado
NannoKit.Theme
Motor de temas unificado com suporte a light/dark mode, paletas customizadas, CSS variables dinâmicas e integração com todos os outros pacotes NannoKit.
Planejado

Todos os pacotes seguem os mesmos padrões de qualidade, documentação e arquitetura do NannoKit.Dialogs.
