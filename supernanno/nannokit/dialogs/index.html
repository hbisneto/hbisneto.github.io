NannoKit.Dialogs

Modern file & message dialogs for SuperNanno and any Textual app.

Clean, priority-aware dialogs with OpenFile, SaveFile, OpenFolder and messagebox. Full keyboard + mouse support.

pip install nannokit-dialogs See Examples →
Textual 8.2+ Python 3.10+ BSD-3-Clause
Features

Dialogs that just work

Built for SuperNanno with seamless integration into any Textual application.

OpenFile & OpenFolder
Multiselect, filters, hidden files toggle, path input and validation.
SaveFile
Overwrite confirmation with smart priority handling.
messagebox
Flexible YES/NO/CANCEL, types (INFO/WARNING/ERROR) and callbacks.
Priority Isolation
messagebox and filedialogs never silently interrupt each other.
Full Keyboard Support
Escape to cancel, Enter to confirm, M to multiselect.
Zero-config Integration
Automatic App detection or explicit attach with DialogManager.
Quick Start

Get up and running in seconds

Installation
$ pip install nannokit-dialogs
Example: File Dialogs
from nannokit.dialogs import OpenFile, SaveFile, OpenFolder

OpenFile.show(
    initial_directory=".",
    filters=["*.py", "*.toml"],
    callback=lambda p: print("Selected:", p)
)

SaveFile.show(default_filename="config.toml", callback=...)
Example: MessageBox
from nannokit.dialogs import messagebox

messagebox.show(
    "Save changes before closing?",
    buttons=messagebox.buttons.YES_NO_CANCEL,
    type=messagebox.type.WARNING,
    callback=...
)
Priority Isolation

Dialogs that respect each other

Thanks to DialogQueue and priority tiers, no dialog can silently interrupt another.
