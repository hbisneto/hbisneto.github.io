Política de Privacidade

Última atualização: 13 de maio de 2026

Esta Política de Privacidade descreve como o software SuperNanno coleta, utiliza, armazena e protege informações relacionadas aos usuários, em conformidade com a Lei Geral de Proteção de Dados Pessoais — Lei nº 13.709/2018 (LGPD).

Ao utilizar o SuperNanno, o usuário declara estar ciente e de acordo com esta Política.

Princípio Fundamental: O SuperNanno foi projetado para operar prioritariamente de forma local, respeitando ao máximo a privacidade e a soberania dos dados do usuário.

1. Identificação do Software

O SuperNanno é um editor de texto moderno baseado em terminal, desenvolvido para edição local de arquivos, com funcionalidades como edição de texto, leitura/gravação de arquivos, restauração de sessão, gerenciamento de backups, busca textual, logs locais e configuração personalizada.

2. Escopo desta Política

Esta Política aplica-se ao SuperNanno, aos arquivos de configuração, sessão, logs locais, backups e futuras funcionalidades de integração.

Não se aplica a softwares de terceiros utilizados em conjunto com o SuperNanno.

3. Dados Coletados

3.1 Dados de Arquivos

Nomes de arquivos, caminhos, conteúdo textual, extensões e metadados necessários para o funcionamento do editor. Utilizados exclusivamente para operação local.

3.2 Dados de Sessão

Último arquivo aberto, lista de arquivos recentes, preferências de sessão e estado do editor — armazenados localmente.

3.3 Dados de Configuração

Preferências visuais, comportamento do editor, diretórios de backup, indentação, pesquisa e demais configurações do usuário.

3.4 Logs Técnicos

Eventos operacionais, mensagens de diagnóstico e informações técnicas armazenadas localmente para melhoria de estabilidade e diagnóstico.

3.5 Dados de CLI

Argumentos de linha de comando (caminhos, termos de busca, etc.) processados apenas durante a execução.

4. Dados Não Coletados

O SuperNanno não coleta dados bancários, senhas, biometria, localização, não realiza rastreamento publicitário, não vende dados e não transmite conteúdo por padrão para servidores externos.

5. Finalidade do Tratamento

Funcionamento do editor, abertura/salvamento de arquivos, restauração de sessão, backups, personalização, diagnóstico técnico e melhoria contínua de estabilidade.

6. Base Legal (LGPD)

  • Execução de Contrato (Art. 7º, V)
  • Legítimo Interesse (Art. 7º, IX) — estabilidade, segurança e diagnóstico
  • Consentimento (Art. 7º, I) — para futuras funcionalidades online

7. Armazenamento de Dados

Todos os dados são armazenados localmente no dispositivo do usuário nos seguintes diretórios:

  • macOS: ~/Library/Application Support/Bisneto/SuperNanno
  • Linux: ~/.config/Bisneto/SuperNanno
  • Windows: %APPDATA%/Bisneto/SuperNanno

8. Compartilhamento de Dados

O SuperNanno não compartilha dados automaticamente. Qualquer compartilhamento ocorre apenas mediante ação explícita do usuário ou obrigação legal.

9. Plugins e Integrações Futuras

Funcionalidades futuras (plugins, sincronização, telemetria) serão claras sobre coleta de dados e, quando necessário, solicitarão consentimento específico.

10. Segurança da Informação

Medidas técnicas alinhadas com aplicações locais: processamento offline prioritário, isolamento de arquivos e minimização de dependências externas.

11. Direitos do Titular (LGPD)

O usuário pode exercer seus direitos (acesso, correção, eliminação, etc.) diretamente através do gerenciamento dos arquivos locais ou por contato com o desenvolvedor.

12. Contato

Para dúvidas sobre privacidade, entre em contato através dos canais oficiais do projeto no GitHub.

Esta Política pode ser atualizada periodicamente. A versão mais recente está sempre disponível nesta página.
