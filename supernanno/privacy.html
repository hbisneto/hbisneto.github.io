✦ Transparency & Data Protection

Privacy Policy

Last Updated: Jun 02, 2026

This Privacy Policy describes how the SuperNanno software collects, uses, stores, and protects information related to users, in compliance with the General Data Protection Law — Law No. 13.709/2018 (LGPD).

By using SuperNanno, the user declares to be aware and in agreement with this Policy.

Fundamental Principle: SuperNanno was designed to operate primarily locally, respecting the user's privacy and data sovereignty to the maximum extent.

1. Software Identification

SuperNanno is a modern terminal-based text editor, developed for local file editing, with features such as text editing, file reading/writing, session restoration, backup management, textual search, local logs, and customized configuration.

2. Scope of this Policy

This Policy applies to SuperNanno, its configuration files, session data, local logs, backups, and future integration features.

It does not apply to third-party software used in conjunction with SuperNanno.

3. Data Collected

3.1 File Data

File names, paths, textual content, extensions, and metadata necessary for the editor's operation. Used exclusively for local operation.

3.2 Session Data

Last opened file, list of recent files, session preferences, and editor state — stored locally.

3.3 Configuration Data

Visual preferences, editor behavior, backup directories, indentation, search, and other user configurations.

3.4 Technical Logs

Operational events, diagnostic messages, and technical information stored locally for stability improvement and diagnostics.

3.5 CLI Data

Command-line arguments (paths, search terms, etc.) processed only during execution.

4. Data Not Collected

SuperNanno does not collect banking data, passwords, biometric data, location information, does not perform advertising tracking, does not sell data, and does not transmit content by default to external servers.

5. Purpose of Data Processing

Editor operation, file opening/saving, session restoration, backups, customization, technical diagnostics, and continuous stability improvement.

6. Legal Basis (LGPD)

  • Contract Execution (Art. 7º, V)
  • Legítimo Interesse (Art. 7º, IX) — estabilidade, segurança e diagnóstico
  • Consentimento (Art. 7º, I) — para futuras funcionalidades online

7. Data Storage

All data is stored locally on the user's device in the following directories:

  • macOS: ~/Library/Application Support/Bisneto/SuperNanno
  • Linux: ~/.config/Bisneto/SuperNanno
  • Windows: %APPDATA%/Bisneto/SuperNanno

8. Data Sharing

SuperNanno does not automatically share data. Any sharing occurs only upon explicit user action or legal obligation.

9. Future Plugins and Integrations

Future features (plugins, synchronization, telemetry) will be clear about data collection and, when necessary, request specific consent.

10. Information Security

Technical measures aligned with local applications: prioritized offline processing, file isolation, and minimization of external dependencies.

11. Rights of the Data Subject (LGPD)

O usuário pode exercer seus direitos (acesso, correção, eliminação, etc.) diretamente através do gerenciamento dos arquivos locais ou por contato com o desenvolvedor.

The user can exercise their rights (access, correction, deletion, etc.) directly through the management of the local files or by contact with the developer.

12. Contact

For questions about privacy, please contact the project through its official channels on GitHub.

This Policy may be updated from time to time. The latest version is always available on this page.
